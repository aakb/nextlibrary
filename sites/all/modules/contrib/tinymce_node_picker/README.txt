// ------------------------------------------------------------
// SUMMARY

Drupal Node Picker for TinyMCE provides a button in TinyMCE that displays a list of links to content.  Choosing a link inserts it into the editor.


// ------------------------------------------------------------
// INSTALLATION

* See INSTALL.txt in this directory.

// ------------------------------------------------------------
// USAGE

Once enabled according to INSTALL.txt.

* Create a piece of content using the TinyMCE editor.
* Click the Drupal icon on the toolbar.
* Choose an existing link.
* The link is displayed in the editor.

// ------------------------------------------------------------
// CUSTOMIZATION

To customize the view displayed by the node picker, navigate to admin/build/views and edit the views nodepicker_nodes
and nodepicker_taxonomy.

// ------------------------------------------------------------
// INTEGRATION WITH PATHFILTER

To integrate Node picker with the PathFilter module follow the steps outlined in this post:
http://drupal.org/node/687650#comment-2510656
